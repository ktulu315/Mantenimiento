# Mantenimiento
Herramientas para mantenimientos de PC

# 1. StopSearch
## Objetivo
Eliminar la molesta busqueda de internet en el menú inicio de Windows 10 y Windows 11.
### Instrucciones
#### Método 1
1. Descargar [archivo](https://gist.github.com/ktulu315/331d1766ce13642b003121bdca05417b/archive/74197a6371c2fc9ba376fed1d9ade6a06e90ed7b.zip)
2. Extraer y ejecutar el archivo .reg
#### Método 2
1. Crear un archivo en blanco.
2. Pegar todo el contenido del texto.
3. Guardar el archivo con extension .reg.
4. Ejecutar.
